# Backlink- & Authority-Strategie: mv-inks.de

**Mavie Sommerfeld | Tattoo-Kuenstlerin | Hamburg**
**Erstellt: 21.03.2026**

---

## Zusammenfassung

mv-inks.de ist eine neue Domain ohne bestehende Authority. Diese Strategie zielt darauf ab, innerhalb von 6-12 Monaten eine solide Domain Authority (DA 15-25) aufzubauen durch eine Kombination aus Verzeichniseintraegen, Content-Marketing, PR und Cross-Promotion.

## 4. Cross-Promotion

### 4.1 Heart Over Head (Resident Studio)

| Aktion | URL/Detail | Prioritaet |
|--------|-----------|------------|
| Kuenstler-Profil auf Studio-Website | https://heartoverhead.hamburg | HOCH |
| Tattoodo Studio-Zuordnung | https://www.tattoodo.com/studios/heart_over_head_1 | HOCH |
| Gegenseitige Verlinkung | mv-inks.de verlinkt auf HoH, HoH verlinkt auf mv-inks.de | HOCH |
| Instagram Cross-Tagging | Jedes Tattoo-Bild: @heartoverhead_tattoo + @mv.inks | LAUFEND |

**Massnahme:** Lilly (Studio-Gruenderin) kontaktieren und darum bitten, Mavies Profil mit Link zu mv-inks.de auf heartoverhead.hamburg einzurichten. Das ist der wertvollste einzelne Backlink, da thematisch und lokal perfekt relevant.

## 6. Umsetzungsplan nach Prioritaet

- [ ] Tattoodo Profil anlegen mit 20+ Bildern
- [ ] Tattooscout Profil erstellen
- [ ] MyTattoo.com Profil erstellen
- [ ] Behance Portfolio mit Website-Link
- [ ] Pinterest Business Account mit Website-Verifizierung

### Phase 2: Kurzfristig (Woche 3-6)

- [ ] Inkstinct Profil erstellen
- [ ] iNKPPL Profil erstellen
- [ ] TattooFilter Profil erstellen
- [ ] Yelp & golocal Eintraege
- [ ] Ersten Blog-Artikel veroeffentlichen ("Erstes Tattoo Guide")
- [ ] Pflege-Checkliste PDF erstellen
- [ ] Pitch an Geheimtipp Hamburg senden
- [ ] Pitch an SZENE HAMBURG senden

### Phase 3: Mittelfristig (Monat 2-3)

- [ ] Feelfarbig-Empfehlung anstreben (ueber andere Kuenstler)
- [ ] Podcast-Pitches an "Augen Zu und Durch" und "Tales From The Needle"
- [ ] 4 weitere Blog-Artikel veroeffentlichen
- [ ] Mandala-Infografik erstellen und auf Pinterest promoten
- [ ] Convention-Anmeldung fuer Leeraner Tattoo Weekend

### Phase 4: Langfristig (Monat 4-12)

- [ ] Tattoo-Magazin-Feature anstreben (TaetowierMagazin, Tattoo Spirit)
- [ ] Lokale Presse-Kontakte aufbauen (Abendblatt, MOPO)
- [ ] Guest-Spots in anderen Staedten fuer Netzwerk-Backlinks
- [ ] Charity-Aktion planen fuer PR-Coverage
